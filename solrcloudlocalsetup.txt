1. Download zookeeper bin tar file
2. Download Solr 9.X version
3. keep them on Slorcloudlocal folder
4. Extract zookeeper and solr
5. Go to zookeeper conf and copy sample zookeeper conf to zoo.cfg
6. Configure Java Path in zkserver.cmd as set Java="c:\program files\java\jdk17"
   Configure zoo.cfg with data dir=<Any path>
   4lw.commands.whitelist=mntr,conf,ruok
 
7. Run zkserver.cmd
8. Go to Solr bin folder
9. set Java path in solrin.cmd
10. Start Solr by giving solr start - c -z localhost:2181
   Go to browser, http://localhost:8983/solr
11. update configuration in zookeeper
   solr zk upconfig -d ../server/solr/configsets/_default -n schemaless -z 127.0.0.1:2181
   copy solrconfig.xml from solr to zookeeper root folder

12. create collection 
solr create -c products -n schemaless -shards 1 -replicationFactor 2
